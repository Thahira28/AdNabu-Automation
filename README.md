# AdNabu Automation Test

## Project Overview
This project contains Selenium automation test scripts for login and add-to-cart functionality using Python.

## Tools Used
- Python
- Selenium
- Pytest
- Pytest HTML Report
- WebDriver Manager

## Setup Instructions

1. Install Python

2. Install dependencies:
pip install -r requirements.txt

3. Run test:
python -m pytest automation.py --html=report.html --self-contained-html

## Test Scenario
- Login to SauceDemo website
- Add product to cart
- Verify product added successfully

## Generated Report
Open report.html after execution.
